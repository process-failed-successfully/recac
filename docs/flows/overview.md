# Agent Flows Overview

This directory documents the various agent flows and state machines within the RECAC system using Mermaid charts.

## Available Flows

- [System Overview](overview.md): High-level architecture of the agent interactions.
- [Execution Loop](execution_loop.md): Detailed flow of the main execution loop in `RunLoop`.
- [Agent States](agent_states.md): State machine diagram showing transitions between Coding, QA, Manager, and Cleaner states.
- [Task Execution](task_execution.md): Low-level task execution logic handled by `DependencyExecutor`.
